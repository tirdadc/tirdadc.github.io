Updating Your Ubuntu Packages

Sep 17th, 2014 3:04 pm

sudo apt-get update
Update source list.

sudo apt-get upgrade
Upgrade all installed packages.

sudo apt-get autoclean
Remove unused .deb files for packages that are no longer installed.
